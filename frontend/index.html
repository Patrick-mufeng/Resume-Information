使用说明

API Key 管理

  1. 选择提供商 — 根据 API Key 来源选择对应 AI 提供商
  2. 填入 Key — 输入 API Key，支持显示/隐藏
  3. 添加 — 自动检测 Key 可用性，绿灯=有效，红灯=无效
  4. 多 Key 轮询 — 添加多个 Key 自动交替使用
  5. 管理 — 随时启用/禁用、删除 Key

方式一：文件夹批量处理（推荐）

  1. 输入文件夹路径（如 D:/简历文件夹）
  2. 点击"扫描并处理"，自动扫描 PDF/Word/图片
  3. PDF/Word 自动转图片，空白页自动删除
  4. 自动创建任务并启动 AI 识别

方式二：手动上传

  1. 新建任务 → 拖拽文件 → 点击开始处理

自定义提取字段

  1. 字段名 = JSON key，显示名称 = CSV 表头
  2. 格式提示帮助 AI 准确输出（如 "11位"）
  3. 修改后立即生效
  4. 点击"重置为默认"恢复预设字段

查看和导出结果

  1. 支持文件名搜索和状态筛选
  2. 点击"详情"查看完整提取结果
  3. 失败记录可"重试"
  4. 点击"导出 CSV"下载
